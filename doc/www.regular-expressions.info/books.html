Regular Expression Reference

  • Basic Regular Expression Syntax Reference. Characters, anchors, alternation and quantifiers.
  • Advanced Regular Expression Syntax Reference. Grouping, modifiers and lookaround.
  • Unicode-Related Regular Expression Syntax Reference. Regex tokens only supported by Unicode-based regular expression engines.
  • Flavor-Specific Regular Expression Syntax Reference. Regex tokens unique to specific regex flavors.
  • Regular Expression Flavor Comparison. Feature matrix of all regex flavors discussed on this website .
  • Replacement Text Reference. Feature matrix of the search-and-replace tokens of all the tools and languages discussed on this website .
